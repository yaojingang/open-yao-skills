# Example Reports

This folder contains generated example reports for `yao-bayesian-skill`.

They are included here so visitors can inspect the final rendered output, not just the skill source code.

Each case includes `html`, `md`, and `pdf` versions.

## Notes

- HTML reports are bilingual and support browser printing.
- PDF reports are exported from the HTML cases for convenient sharing.
- Newer reports include the prior-hygiene section that surfaces the most relevant everyday Bayesian principles for the case.
- These files are examples of generated output, while the reusable logic still lives in the skill source itself.
